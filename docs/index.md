## Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hrpavi7/My-Daily-Planner.git
   cd My-Daily-Planner
   ```

2. **Install dependencies**
   ```bash
   bun install
   ```

3. **Start development server**
   ```bash
   bun run dev
   ```

4. **Open your browser** to `http://localhost:8080` (or `localhost:5173` in some cases)
